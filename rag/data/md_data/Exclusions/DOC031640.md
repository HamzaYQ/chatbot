# Exclusions liées à l'usage du véhicule — édition 2

## Introduction

Cette deuxième édition des exclusions liées à l'usage du véhicule actualise les informations sur les situations exclues de la garantie. Les évolutions des usages (VTC, livraison, etc.) ont conduit à préciser certaines exclusions qu'il est important de connaître.

## Usage professionnel non déclaré — précisions

L'usage professionnel du véhicule doit être déclaré à l'assureur.

Un usage professionnel non déclaré entraîne l'exclusion de la garantie.

Le transport de marchandises pour le compte d'un tiers est un usage professionnel.

Les déplacements professionnels réguliers doivent être déclarés.

L'usage professionnel occasionnel peut être couvert sous conditions.

## Usage commercial non déclaré — précisions

L'usage commercial du véhicule doit être déclaré à l'assureur.

Le transport de personnes contre rémunération est un usage commercial.

Les services de VTC ou de taxi doivent être déclarés.

Les livraisons de marchandises pour le compte d'un tiers sont commerciales.

L'usage commercial occasionnel peut être couvert sous conditions.

## Usage du véhicule pour des activités illégales — précisions

L'usage du véhicule pour des activités illégales est une exclusion.

Les activités criminelles ou délictueuses ne sont pas couvertes.

Le transport de marchandises illégales est également exclu.

L'usage du véhicule pour des activités de jeux d'argent est exclu.

L'usage du véhicule pour des activités de trafic est exclu.

## Usage du véhicule pour des compétitions — précisions

L'usage du véhicule pour des compétitions est une exclusion.

Les courses, rallyes et autres compétitions sportives ne sont pas couvertes.

Les essais de performance sur circuit sont également exclus.

Les compétitions amateurs sont également exclues.

Les entraînements sur circuit sont également exclus.

## Usage du véhicule pour le transport de matières dangereuses — précisions

Le transport de matières dangereuses est une exclusion.

Les matières explosives, inflammables ou toxiques sont exclues.

Les matières radioactives sont également exclues.

Le transport de déchets dangereux est exclu.

L'usage du véhicule pour le transport de carburant est exclu.

## Usage du véhicule pour le transport de personnes — précisions

Le transport de personnes contre rémunération est une exclusion.

Les services de taxi, VTC ou transport scolaire sont exclus.

Le transport de personnes dans le cadre d'une activité associative peut être couvert.

Le transport de personnes en covoiturage peut être couvert sous conditions.

Le transport de personnes à titre gratuit est couvert.

## Usage du véhicule pour le transport de marchandises — précisions

Le transport de marchandises pour le compte d'un tiers est une exclusion.

Le transport de marchandises dangereuses est également exclu.

Le transport de marchandises pour son propre compte peut être couvert.

Le transport de marchandises à titre gratuit est couvert.

Le transport de marchandises occasionnel peut être couvert sous conditions.

## Conséquences pour l'assuré

En cas d'exclusion, l'assureur n'est pas tenu d'indemniser le sinistre.

L'assuré doit prendre en charge l'intégralité des dommages.

Le contrat peut être résilié par l'assureur en cas d'exclusion.

L'assuré peut être poursuivi par les autorités en cas d'infraction.

Les conséquences peuvent être financières et pénales.

## Recommandations

Déclarer l'usage réel du véhicule à l'assureur.

Ne pas utiliser le véhicule pour des activités illégales.

Ne pas utiliser le véhicule pour des compétitions.

Ne pas transporter de matières dangereuses sans autorisation.

En cas de doute, contacter son assureur pour vérifier la couverture.
