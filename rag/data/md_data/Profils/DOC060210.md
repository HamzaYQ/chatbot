# Profil conducteur : conducteur de véhicule utilitaire

## Présentation du profil

Ce profil concerne les conducteurs de véhicules utilitaires légers (fourgons, pick-ups, camionnettes). Ils utilisent leur véhicule pour le transport de marchandises, de matériel ou pour des activités professionnelles.

## Caractéristiques principales

Âge : généralement entre 30 et 60 ans.

Véhicule : utilitaire léger (fourgon, pick-up, camionnette).

Usage : professionnel ou mixte (professionnel et personnel).

Kilométrage : souvent élevé (plus de 20 000 km par an).

Conduite : régulière, avec des déplacements fréquents.

## Enjeux pour l'assurance

**Usage professionnel** : l'usage professionnel doit être déclaré à l'assureur.

**Chargement** : le transport de marchandises doit être déclaré.

**Kilométrage** : souvent plus élevé que la moyenne.

**Risque** : les utilitaires sont plus exposés aux accidents.

**Franchise** : souvent plus élevée pour les utilitaires.

## Recommandations de garanties

**Responsabilité civile** : obligatoire, avec un plafond adapté.

**Vol** : recommandé si le véhicule est stationné dans des zones non sécurisées.

**Bris de glace** : utile pour les longs trajets.

**Assistance 0 km** : indispensable en cas de panne avec chargement.

**Protection juridique** : recommandée pour les litiges professionnels.

## Packs recommandés

**Pack Business** : spécialement conçu pour les professionnels.

**Pack Gold+** : pour une protection complète.

**Pack VIP** : pour les véhicules de grande valeur.

**Option assistance renforcée** : avec prise en charge du chargement.

**Option véhicule de remplacement** : indispensable pour les professionnels.

## Conseils pour les conducteurs d'utilitaires

Déclarer l'usage professionnel à l'assureur.

Choisir une franchise adaptée à votre budget.

Conserver les factures d'achat et d'entretien.

Souscrire une assistance pour le dépannage du véhicule chargé.

Vérifier les exclusions liées au transport de marchandises.

## Particularités

L'usage professionnel doit être déclaré à l'assureur.

Le transport de marchandises dangereuses est exclu.

Le chargement doit être conforme aux normes.

Les modifications du véhicule doivent être déclarées.

Le kilométrage annuel doit être estimé avec précision.
