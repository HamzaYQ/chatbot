# Profil conducteur : conducteur de véhicule de collection

## Présentation du profil

Ce profil concerne les conducteurs de véhicules de collection (plus de 30 ans, conservés dans leur état d'origine). Ils sont souvent des passionnés d'automobile qui recherchent une assurance adaptée à la valeur spécifique de leur véhicule.

## Caractéristiques principales

Âge : généralement entre 40 et 70 ans.

Véhicule : de collection (plus de 30 ans, conservé dans son état d'origine).

Usage : occasionnel (sorties, expositions, rallyes).

Kilométrage : très faible (moins de 5 000 km par an).

Conduite : prudente, le véhicule est souvent entretenu avec soin.

## Enjeux pour l'assurance

**Valeur spécifique** : la valeur d'un véhicule de collection est souvent supérieure à sa valeur vénale.

**Expertise** : nécessaire pour évaluer la valeur du véhicule.

**Pièces de rechange** : rares et souvent chères.

**Garages spécialisés** : nécessité de garages spécialisés pour les réparations.

**Stockage** : le véhicule doit être stocké dans un lieu sécurisé.

## Recommandations de garanties

**Valeur d'expertise** : indispensable pour une indemnisation à la valeur réelle.

**Vol** : recommandé avec un plafond adapté à la valeur du véhicule.

**Bris de glace** : utile pour les sorties.

**Assistance 0 km** : indispensable en cas de panne.

**Protection juridique** : recommandée en cas de litige.

## Packs recommandés

**Pack VIP** : offre une couverture complète avec une valeur d'expertise.

**Pack Gold+** : pour une protection renforcée.

**Option valeur d'expertise** : recommandée pour les véhicules de collection.

**Option protection juridique** : recommandée en cas de litige.

**Option assistance renforcée** : avec prise en charge des pièces rares.

## Conseils pour les conducteurs de véhicules de collection

Faire expertiser le véhicule pour déterminer sa valeur.

Souscrire une assurance avec une valeur d'expertise.

Choisir un garage spécialisé pour les réparations.

Stocker le véhicule dans un lieu sécurisé.

Ne pas utiliser le véhicule pour des compétitions.

## Particularités

Les véhicules de collection ont une valeur spécifique.

Les pièces de rechange sont rares et chères.

Les réparations nécessitent des garages spécialisés.

Les modifications doivent être déclarées à l'assureur.

Les compétitions sont exclues de la garantie.
