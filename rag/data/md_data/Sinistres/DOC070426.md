# Procédure recours contre l'assureur adverse

## Introduction

Lorsque le sinistre est causé par un tiers, votre assureur peut engager un recours contre l'assureur adverse pour récupérer les sommes versées. Ce guide explique les étapes de cette procédure.

## Qu'est-ce qu'un recours contre l'assureur adverse ?

Le recours est une action engagée par votre assureur contre l'assureur du tiers responsable.

Il permet de récupérer les sommes versées à son assuré.

La procédure est encadrée par le Code des assurances marocain.

Elle peut être amiable ou judiciaire.

L'assuré non responsable n'est pas pénalisé.

## Conditions du recours

Le sinistre doit être causé par un tiers responsable.

L'assuré doit être couvert par une garantie responsabilité civile.

L'assureur doit avoir indemnisé son assuré.

La responsabilité du tiers doit être établie.

Le recours doit être exercé dans les **2 ans** suivant le sinistre.

## Procédure de recours amiable

**Étape 1** : L'assureur identifie l'assureur du tiers responsable.

**Étape 2** : L'assureur envoie une notification de recours.

**Étape 3** : Les deux assureurs échangent les informations (constat, expertises).

**Étape 4** : Un accord est trouvé sur le montant du remboursement.

**Étape 5** : L'assureur du tiers rembourse les sommes versées.

## Procédure de recours judiciaire

**Étape 1** : Si le recours amiable échoue, une procédure judiciaire est engagée.

**Étape 2** : L'assureur assigne l'assureur adverse en justice.

**Étape 3** : Le tribunal examine les preuves.

**Étape 4** : Le tribunal rend une décision sur la responsabilité.

**Étape 5** : L'assureur adverse est condamné à rembourser.

## Délais de prescription

Le recours doit être exercé dans les **2 ans** suivant le sinistre.

Ce délai court à compter de la date du sinistre.

La prescription peut être interrompue par une action en justice.

La prescription peut être suspendue en cas de force majeure.

Passé ce délai, le recours n'est plus possible.

## Conséquences pour l'assuré

L'assuré non responsable n'est pas pénalisé.

Son bonus-malus n'est pas affecté.

L'assureur se charge de toutes les démarches.

L'assuré doit collaborer avec l'assureur.

Il n'a pas à avancer les frais de réparation.

## Recommandations

Transmettez tous les documents à votre assureur.

Collaborez avec votre assureur dans les démarches.

Ne reconnaissez pas votre responsabilité sans l'accord de votre assureur.

En cas de doute, contactez votre assureur.

Conservez tous les documents relatifs au sinistre.
